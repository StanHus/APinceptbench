# AP Question Types

Reference: [College Board AP Central](https://apcentral.collegeboard.org/)

## Supported Courses

| Course | Types |
|--------|-------|
| APUSH / APWH / APEH | MCQ, SAQ, DBQ, LEQ |
| APGOV | MCQ, FRQ (4 types) |
| APHG | MCQ, FRQ (3 types) |

## History Question Types

### MCQ (Multiple Choice)
- 4 options (A-D), often in stimulus sets
- **Critical**: factual accuracy, stimulus alignment, answer validity
- **Scoring**: 1 point per correct

### SAQ (Short Answer)
- 3 parts: (a), (b), (c)
- **Critical**: prompt clarity, historical specificity
- **Scoring**: 0-3 points (1 per part)

### DBQ (Document-Based)
- Prompt + **7 documents**
- **Critical**: document count, diversity, authenticity, prompt arguability
- **Scoring**: 0-7 points

| Category | Points |
|----------|--------|
| Thesis | 0-1 |
| Contextualization | 0-1 |
| Evidence (docs) | 0-2 |
| Outside evidence | 0-1 |
| Sourcing (HIPP) | 0-1 |
| Complexity | 0-1 |

### LEQ (Long Essay)
- Prompt requiring thesis, no documents
- Reasoning types: `causation`, `comparison`, `ccot`
- **Critical**: prompt arguability, reasoning type, time period
- **Scoring**: 0-6 points

## Task Verbs

| Verb | Points |
|------|--------|
| Identify / Define / Describe | 1 |
| Explain | 1-2 |
| Compare / Analyze / Evaluate | 2 |

## Evaluation Checklist

1. **Structure**: correct format, required components
2. **Accuracy**: facts, dates, names correct
3. **Skill alignment**: appropriate AP skill tested
4. **Rubric alignment**: clear path to points
5. **Curriculum alignment**: within course scope
