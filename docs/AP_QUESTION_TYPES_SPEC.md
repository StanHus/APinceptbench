# AP Social Studies Question Types Specification

**Generated:** 2026-02-24
**Source:** [College Board AP Central](https://apcentral.collegeboard.org/)

This document defines the official AP question types, rubrics, and evaluation criteria for the AP Social Studies evaluator.

| Course | Code | Question Types |
|--------|------|----------------|
| AP U.S. History | APUSH | MCQ, SAQ, DBQ, LEQ |

## Question Type Definitions

### 1. MCQ - Multiple Choice Question

**Applies to:** All AP Social Studies courses

**Structure:**
- 4-5 answer options (A-D or A-E)
- Often presented in **stimulus sets** (3-4 questions per stimulus)
- Stimulus can be: primary source, secondary source, image, map, graph, table, political cartoon

**Scoring:** 1 point per correct answer

**Evaluation Dimensions:**
| Dimension | Weight | Criteria |
|-----------|--------|----------|
| `factual_accuracy` | Critical | All historical claims must be accurate |
| `stimulus_alignment` | Critical | Question must relate to provided stimulus |
| `answer_validity` | Critical | Exactly one correct answer, defensible |
| `distractor_quality` | High | Plausible but clearly incorrect |
| `skill_alignment` | High | Tests appropriate AP skill (sourcing, causation, comparison, etc.) |
| `difficulty_calibration` | Medium | Matches requested difficulty level |

---

### 2. SAQ - Short Answer Question

**Applies to:** APUSH, APWH, APEH (AP History courses)

**Structure:**
- 3 parts: (a), (b), (c)
- Some include stimulus (document, image, data)
- Response: 3-4 sentences per part
- Tests specific historical skills

**Scoring:** 0-3 points (1 point per part)

**Official Rubric:**
| Part | Points | Requirement |
|------|--------|-------------|
| (a) | 0-1 | Accomplishes task A with specific historical evidence |
| (b) | 0-1 | Accomplishes task B with specific historical evidence |
| (c) | 0-1 | Accomplishes task C with specific historical evidence |

**Evaluation Dimensions:**
| Dimension | Weight | Criteria |
|-----------|--------|----------|
| `prompt_clarity` | Critical | Each part clearly defines task |
| `historical_specificity` | Critical | Requires specific evidence, not generalizations |
| `skill_alignment` | High | Tests causation, comparison, or contextualization |
| `stimulus_relevance` | High | If stimulus present, parts reference it |
| `answerability` | High | Parts can be answered with course content |
| `scoring_criteria` | High | Clear what earns the point |

---

### 3. DBQ - Document-Based Question

**Applies to:** APUSH, APWH, APEH (AP History courses)

**Structure:**
- 1 prompt requiring argumentative thesis
- **7 documents** (various types: text, image, map, chart, political cartoon)
- Documents represent multiple perspectives
- 60 minutes (including 15-min reading period)

**Scoring:** 0-7 points

**Official Rubric (2024):**
| Category | Points | Requirement |
|----------|--------|-------------|
| **Thesis/Claim** | 0-1 | Historically defensible thesis establishing line of reasoning |
| **Contextualization** | 0-1 | Describes broader historical context relevant to prompt |
| **Evidence (Documents)** | 0-2 | Uses 3+ docs (1pt) OR 4+ docs supporting argument (2pts) |
| **Evidence (Outside)** | 0-1 | Uses specific historical evidence beyond the documents |
| **Sourcing (HIPP)** | 0-1 | Explains how/why 2+ docs' POV, purpose, situation, or audience is relevant |
| **Complexity** | 0-1 | Sophisticated argument OR uses all 7 docs + HIPP on 4 docs |

**Document Requirements:**
- 7 documents total
- Mix of document types (primary, secondary, visual, quantitative)
- Multiple perspectives represented
- Historically authentic or accurately simulated
- Proper attribution (author, date, source)

**Evaluation Dimensions:**
| Dimension | Weight | Criteria |
|-----------|--------|----------|
| `prompt_quality` | Critical | Requires arguable thesis with multiple valid positions |
| `document_count` | Critical | Exactly 7 documents |
| `document_diversity` | Critical | Multiple types and perspectives |
| `document_authenticity` | Critical | Historically accurate or properly attributed |
| `document_relevance` | Critical | All docs relevant to prompt |
| `thesis_potential` | High | Multiple defensible thesis positions possible |
| `outside_evidence_scope` | High | Clear what outside evidence applies |
| `time_period_alignment` | High | Matches specified historical period |
| `sourcing_opportunity` | Medium | Documents allow HIPP analysis |

### 4. LEQ - Long Essay Question

**Applies to:** APUSH, APWH, APEH (AP History courses)

**Structure:**
- 1 prompt requiring argumentative thesis
- No documents provided (relies on student knowledge)
- Tests specific historical reasoning skill
- 40 minutes recommended
- Students typically choose from 3 prompts (different time periods)

**Scoring:** 0-6 points

**Official Rubric (2024):**
| Category | Points | Requirement |
|----------|--------|-------------|
| **Thesis/Claim** | 0-1 | Historically defensible thesis establishing line of reasoning |
| **Contextualization** | 0-1 | Describes broader historical context relevant to prompt |
| **Evidence** | 0-2 | Identifies specific evidence (1pt) OR uses evidence to support argument (2pts) |
| **Analysis & Reasoning** | 0-2 | Uses historical reasoning (1pt) OR demonstrates complexity (2pts) |

**Historical Reasoning Types:**
- **Causation:** Explain causes and/or effects
- **Comparison:** Compare and/or contrast
- **Continuity and Change (CCOT):** Analyze patterns over time

**Evaluation Dimensions:**
| Dimension | Weight | Criteria |
|-----------|--------|----------|
| `prompt_arguability` | Critical | Requires arguable thesis |
| `reasoning_type` | Critical | Clearly requires causation, comparison, or CCOT |
| `time_period_specificity` | Critical | Clear time boundaries |
| `evidence_accessibility` | High | Students can cite specific evidence from course |
| `complexity_opportunity` | High | Allows for nuanced argument |
| `skill_alignment` | High | Tests appropriate AP skill |

### 5. AP Government FRQs (4 Types)

**Applies to:** APGOV only

#### 5a. Concept Application FRQ
**Structure:**
- Political scenario presented
- 3-4 parts asking students to describe, explain, and apply concepts
- Tests application of course concepts to real situations

**Scoring:** Varies by question (typically 3-4 points)

**Evaluation Dimensions:**
| Dimension | Weight | Criteria |
|-----------|--------|----------|
| `scenario_realism` | Critical | Plausible political scenario |
| `concept_integration` | Critical | Tests specific course concepts |
| `part_clarity` | High | Each part has clear task |
| `application_requirement` | High | Requires application, not just recall |

#### 5b. Quantitative Analysis FRQ
**Structure:**
- Visual data (chart, graph, table, infographic)
- 3-4 parts analyzing data and connecting to political concepts

**Scoring:** Varies by question (typically 4 points)

**Evaluation Dimensions:**
| Dimension | Weight | Criteria |
|-----------|--------|----------|
| `data_accuracy` | Critical | Data must be accurate/realistic |
| `data_readability` | Critical | Clear, interpretable visualization |
| `analysis_requirement` | High | Requires trend identification and explanation |
| `concept_connection` | High | Links data to political principles |

#### 5c. SCOTUS Comparison FRQ
**Structure:**
- Non-required Supreme Court case presented
- Students compare with a required foundational case
- Tests understanding of constitutional principles

**Required Cases (must know):**
Marbury v. Madison, McCulloch v. Maryland, Schenck v. United States, Brown v. Board, Engel v. Vitale, Baker v. Carr, Gideon v. Wainwright, Tinker v. Des Moines, NY Times v. United States, Wisconsin v. Yoder, Roe v. Wade, Shaw v. Reno, United States v. Lopez, McDonald v. Chicago, Citizens United v. FEC

**Scoring:** Typically 4 points

**Evaluation Dimensions:**
| Dimension | Weight | Criteria |
|-----------|--------|----------|
| `case_accuracy` | Critical | Non-required case facts are accurate |
| `required_case_relevance` | Critical | Clear connection to a required case |
| `constitutional_principle` | High | Tests understanding of legal principles |
| `comparison_validity` | High | Meaningful comparison possible |

#### 5d. Argument Essay FRQ
**Structure:**
- Prompt requiring thesis/claim
- Must use evidence from foundational documents
- 6-point rubric similar to LEQ

**Foundational Documents:**
Declaration of Independence, Articles of Confederation, Constitution, Federalist Papers (10, 51, 70, 78), Brutus No. 1, Letter from Birmingham Jail, etc.

**Scoring:** 0-6 points

**Official Rubric:**
| Category | Points | Requirement |
|----------|--------|-------------|
| **Thesis/Claim** | 0-1 | Defensible claim establishing line of reasoning |
| **Evidence** | 0-3 | Uses specific evidence from foundational docs and course concepts |
| **Reasoning** | 0-1 | Explains relationship between evidence and argument |
| **Rebuttal** | 0-1 | Responds to opposing perspective with refutation |

### 6. AP Human Geography FRQs (3 Types)

**Applies to:** APHG only

**Common Structure:**
- Each FRQ worth 7 points
- Tests geographic concepts, processes, and models
- 25 minutes recommended per question

#### 6a. FRQ Type 1 (No Stimulus)
- Pure knowledge-based question
- Tests recall and application of geographic concepts

#### 6b. FRQ Type 2 (One Stimulus)
- Includes map, image, data table, or chart
- Students analyze stimulus and apply concepts

#### 6c. FRQ Type 3 (Two Stimuli)
- Two related visuals/data sources
- Requires comparison and synthesis

**Evaluation Dimensions (all types):**
| Dimension | Weight | Criteria |
|-----------|--------|----------|
| `stimulus_quality` | Critical | Clear, readable, geographically accurate |
| `geographic_concept` | Critical | Tests specific APHG concepts |
| `part_clarity` | High | Each part has clear task verb |
| `spatial_thinking` | High | Requires geographic reasoning |
| `scale_appropriateness` | Medium | Appropriate geographic scale |

## Task Verb Definitions

AP FRQs use specific task verbs that determine how responses are scored:

| Verb | Definition | Points |
|------|------------|--------|
| **Identify** | Provide specific information | 1 |
| **Define** | Provide meaning of a term | 1 |
| **Describe** | Provide relevant characteristics | 1 |
| **Explain** | Provide information about how or why | 1-2 |
| **Compare** | Provide description of similarities AND differences | 2 |
| **Analyze** | Examine components and their relationships | 2 |
| **Evaluate** | Assess strengths and weaknesses | 2 |

## Evaluator Implementation Requirements

### For Each Question Type, Evaluate:

**1. Structural Validity**
- Correct format for question type
- Required components present (stimulus, documents, parts)
- Proper attribution and sourcing

**2. Content Accuracy**
- Historical/geographic facts correct
- Dates, names, events accurate
- No anachronisms

**3. Skill Alignment**
- Tests appropriate AP skill
- Matches specified reasoning type
- Appropriate cognitive demand

**4. Rubric Alignment**
- Clear path to earning each rubric point
- Scoring criteria are unambiguous
- Model responses achievable

**5. Curriculum Alignment**
- Content matches course framework
- Time period appropriate
- Topic within scope
